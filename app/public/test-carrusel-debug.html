Abre la consola (F12) y luego ve a /anuncios/

Este script te ayudará a diagnosticar el problema del carrusel.

Copia este código y pégalo en la consola del navegador cuando estés en /anuncios/:

// Test manual del carrusel
const nextBtn = document.getElementById('modernNextBtn');
const prevBtn = document.getElementById('modernPrevBtn');

console.log('Next button:', nextBtn);
console.log('Prev button:', prevBtn);
console.log('Next disabled:', nextBtn?.disabled);
console.log('Prev disabled:', prevBtn?.disabled);

// Forzar clic
if (nextBtn) {
    nextBtn.click();
    console.log('Click en NEXT ejecutado');
}
