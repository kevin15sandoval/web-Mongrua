Descargue el siguiente archivo que contiene un documento .html con el código necesario para incluir el formulario de solicitud de usuario y contraseña en su web. La persona o entidad encargada de administrar su web podrá acceder al código HTML del documento descargado, cortarlo y colocarlo en cualquier parte de su web (donde desee que se visualice el formulario de acceso). También puede personalizar el formulario con su imagen corporativa modificando tipos de letra, añadiendo imágenes, etc. Funcionará correctamente siempre y cuando no se modifique el nombre de las variables y la dirección de destino.
